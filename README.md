## 개발환경
- Visual Studio Code

## 프로젝트 개요
- CSS를 사용하여 기본 navigation을 만들고 로그인 링크를 작성
- 로그인 링크를 누르면 로그인 창이 팝업처럼 생기게 구현.
- javascript에 밖에 화면을 클릭하면 로그인 창이 저절로 꺼지는 function을 구현.
